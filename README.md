# 🪐 Cosmos Explorer

An interactive guide to the Solar System. Tap a planet to reveal real stats (diameter, day length, distance, moons), then test yourself with a six-question quiz. Starfield background rendered purely in CSS.

**Live demo:** https://skrizidev.github.io/cosmos-explorer/

## Pages
- **Explore** — clickable planet orrery with a detail panel
- **Quiz** — instant-feedback space quiz with scoring
- **About** — data notes

## Tech
Vanilla HTML / CSS / JavaScript. No libraries.
